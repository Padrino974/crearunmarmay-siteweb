# Crearun Marmay — Site web

Site statique prêt à publier sur GitHub Pages.

## Fichiers

- `index.html` : page principale du site
- `.nojekyll` : évite le traitement Jekyll par GitHub Pages

## Publier sur GitHub Pages

1. Crée un nouveau dépôt GitHub, par exemple `crearunmarmay-site`.
2. Envoie les fichiers de ce dossier dans le dépôt.
3. Va dans **Settings > Pages**.
4. Dans **Build and deployment**, choisis :
   - **Source** : Deploy from a branch
   - **Branch** : `main`
   - **Folder** : `/root`
5. Clique sur **Save**.

Ton site sera disponible à une adresse du type :

`https://ton-utilisateur.github.io/crearunmarmay-site/`

## Modifier le site

Tu peux modifier directement `index.html` pour changer les textes, images, liens Beacons ou coordonnées.
